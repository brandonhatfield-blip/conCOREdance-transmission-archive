# Print to ConCOREdance — Universal Clinical Document Import

## Summary

ConCOREdance establishes Print to ConCOREdance as a signature universal import feature, allowing therapists to receive PDFs from browsers, EHRs, and other printable macOS applications into a fast encrypted review tray.

## Body

Brandon identified a foundational portability insight: instead of requiring a custom integration with every EHR, ConCOREdance can use the printing capability those systems already share.

Print to ConCOREdance is therefore established as a signature product feature and universal fallback migration path. A therapist should be able to print from a browser, EHR, office application, or other macOS software; route the resulting PDF into ConCOREdance; confirm the client, category, and document date; and file the preserved original.

The first native prototype now includes a Document Import workspace supporting PDF file selection, drag-and-drop, and files opened with ConCOREdance. Incoming originals are encrypted into a separate attachment store, indexed in the encrypted vault, checked for duplicate content by SHA-256, and held in an awaiting-review state until the therapist explicitly files them.

The intended macOS distribution path will add ConCOREdance to the Print dialog's PDF workflows so the same import review tray can receive documents directly from printing. The supported system path is preferred over a custom printer driver or raw spool interception.

Speed is part of the doctrine. The product target is one document in under 15 seconds and an ordinary client chart in under five minutes. The import experience must not recreate vendor captivity with prettier surfaces.

## Decisions

- Print to ConCOREdance is a signature public-facing product feature.
- Printing, Open With, drag-and-drop, and file selection feed one encrypted import review tray.
- Imported originals are preserved unchanged before encryption.
- Duplicate content is detected before a second encrypted copy is created.
- Nothing is permanently filed without therapist confirmation.
- The supported macOS PDF workflow is preferred over custom printer-driver or raw spool interception.
